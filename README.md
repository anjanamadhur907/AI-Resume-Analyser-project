# AI Resume Analyzer

An AI-powered Resume Analyzer built with **Python, FastAPI, SQLAlchemy, MySQL, Jinja2, and Google Gemini AI**. The application allows users to securely upload PDF resumes, extract text, generate ATS scores, identify skills, analyze strengths and weaknesses, and receive AI-powered recommendations through an interactive dashboard.

## Features

* User authentication and session management
* PDF resume upload and text extraction
* AI-powered resume analysis using Google Gemini AI
* ATS score and overall resume evaluation
* Automatic skill extraction
* Strengths, weaknesses, and improvement recommendations
* Resume history dashboard
* Analysis reports stored in MySQL using SQLAlchemy ORM
* Modular Repository-Service architecture with FastAPI

## Tech Stack

* Python
* FastAPI
* SQLAlchemy
* MySQL
* Jinja2
* Google Gemini AI
* Bootstrap
